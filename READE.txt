This tool uses **Bootstrap Icons** to create a customizable and practical **icon box**.

* You can customize the **layout and design** of the icon box using the `icon-settings.css` file according to your project‚Äôs theme.

### 6Ô∏è‚É£ Add buttons for icons

Inside the `.box`, create as many `<button>` elements as you need.
Each button must have **two classes**:

1. `icon`
2. the icon name (for example: `whatsapp`, `instagram`, `telegram`)

Example:

```html

  <div class="box">
    <button class="icon whatsapp"></button>
    <button class="icon instagram"></button>
    <button class="icon telegram"></button>
  </div>
</section>
```
